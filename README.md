### Featured Project

**[Sistema de Revision de Operaciones](https://github.com/ISoledadSS/sis-rev-ops)** — A multi-user web application for managing the full review cycle of securities buy/sell operations. Built with Streamlit + PostgreSQL. Handles concurrent users with contract-level locking, automated position validation, real-time market data integration (Yahoo Finance), interactive Plotly dashboards, PDF/email reporting, and full audit trails. Currently in production processing 800+ daily operations.
